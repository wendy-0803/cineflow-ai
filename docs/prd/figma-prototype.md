## 原型内容

这版原型是灰阶低保真线框，重点表达：

- 从工作台进入新建项目。
- 用户输入一句创意和基础参数。
- 系统先生成故事方案和角色卡。
- 用户确认后进入分镜编辑。
- 分镜页支持镜头编辑、提示词复制和局部重生成入口。
- 导出页支持 Markdown / JSON / PDF 和下游工具提示词包。
- 历史项目页支持打开、复制、版本历史和删除。
