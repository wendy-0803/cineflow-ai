## 适合面试时怎么讲

这个项目可以概括为一句话：

> 我做了一个面向短视频创作者的 AI 短片脚本生成与分镜规划工具，把从一句创意到故事梗概、三幕结构、角色设定、分场景脚本、分镜、提示词和导出方案的链路产品化。
